# Project Brief: CapMan AI

## Overview
AI-driven gamified scenario training and MTSS agent for trading education.

## Goals
- Automate and scale deep-reflection, high-repetition trading scenario training
- Generate high volumes of unique, complex trading scenarios without latency
- Gamify the learning experience with XP, leaderboards, and head-to-head challenges
- Provide MTSS (Multi-Tier System of Supports) intervention data for educators
- Enforce proprietary CapMan lexicon and specialized options trading logic via RAG

## Requirements
- Dynamic Scenario Generator: AI creates varied prompts requiring trade analysis (not static questions)
- Probing & Grading Agent: AI asks follow-up questions and grades reasoning, not just answers
- Gamification Layer: XP systems, leaderboards, competitive head-to-head challenges
- Peer Review Module: Users evaluate each other's responses
- MTSS Reporting: Automated classification into support tiers based on performance
- Educator "God View" dashboard: Real-time learner grouping by skill level
- Atlas (Python) integration for CapMan-relevant tooling
- Multi-user engine with real-time interaction support
